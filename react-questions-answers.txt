What is prop drilling in React?
ğŸ”¹ It refers to the process of passing data down the component tree through multiple layers of nested components. - d
ğŸ”¹ It occurs when a piece of data needs to be passed from a higher-level parent component to a lower-level child component that is not directly connected by parent-child relationships. - d
ğŸ”¹ It is also called "lifting state up" or "threading props" - d
ğŸ”¹ But, while doing so, other components have access to the data or props as we are passing it down the chain. - d 
ğŸ”¹ This would result in unnecessary exchange of data to the component which don't really need it. - d 
ğŸ”¹ To avoid this we can use possible solutions like React context API, Redux etc. - d
